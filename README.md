## Quick start
Configure host and port in ```package.json```:  
```js
"host": "adm-dev-kit"	// Default host
"port": 3000		// Default port
```
Install dependencies:
```js
npm i       // install all npm pacakges
jspm i      // install other dependencies
```

Start server
```js
npm start
```
